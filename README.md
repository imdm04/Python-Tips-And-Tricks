# Python Tricks and Tips

## 1) GCD
> this python code is for find gcd using recursive mannor.
```
def GCD(A,B):
  if B == 0:
    return A
  else:
    return GCD(B,A%B)

print(GCD(6,12)) 
```
> Output:   6
